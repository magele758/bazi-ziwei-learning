# 紫微斗数 Web · 全量排盘

## 原理与实现

本目录使用 **py-iztro**（封装自 **iztro** JavaScript 库）输出命盘数据，涵盖常见「专业排盘」中的：

- 十二宫、宫干宫支  
- 十四主星及**庙旺平陷**、**四化**（化禄权科忌）  
- 辅星、杂曜  
- 长生十二神、博士十二神、将前十二神、岁前十二神  
- 大限区间、小限岁数  
- 命主、身主、身宫地支、来因宫（iztro 字段）等  

## 使用注意

1. **必须选择性别**（安星规则依赖阴阳顺逆）。  
2. **时辰**：0点为早子时、23 点为晚子时，其余每 2 小时为一时辰，与 `iztro` 的 `time_index` 一致。  
3. 环境依赖 **pythonmonkey**（由 py-iztro 引入），若安装失败请参阅 [py-iztro](https://github.com/x-haose/py-iztro)。

## 与八字项目的区别

紫微以**十二宫 + 星曜网络**为主；八字以**四柱干支**为主。二者在本仓库中为独立应用。
