紫微这一张表，用最笨的话怎么说？

  • 十二宫：把人生切成十二类题目——自己、配偶、钱、工作、健康、朋友……每格一个主题。一眼看懂：想象成十二个文件夹，每个文件夹只讨论一类人生题目；整张盘=十二个格子拼成一圈。
  • 宫干支：这一格的天干地支「门牌号」，用来安星、论生克。一眼看懂：没有这门牌，星就不知道该「落户」在哪；后面看星与星的生克会用到天干。
  • 主星：主角级的星；后面方括号里是庙旺平陷，可理解成这颗星在这个格子里「发挥好还是憋屈」的粗略档位。一眼看懂：主星像主演；庙旺像舞台灯光明亮，陷像灯暗——仍是符号，不是人品判决。
  • 化禄权科忌：四化，像给星加「增益、掌权、名声、纠缠/压力」的滤镜。一眼看懂：禄偏顺、权偏硬、科偏名、忌偏卡——用来描述张力，不是单字定生死。
  • 辅星 / 杂曜：配角与小标签，补细节、补人事象。一眼看懂：主星定基调，辅星杂曜像弹幕评论，补一句「还有这档子事」。
  • 长生12 / 博士12 / 将前12 / 岁前12：四套「神煞/神将」类符号，古法里当吉凶参考，现代多当文化注脚。一眼看懂：四套不同「标签系统」，初学当附录；不必先背全名。
  • 大限：这一宫在哪些岁数段当「十年主角」；小限岁数：与该宫相关的一组年龄提示。一眼看懂：大限=这格宫位「当主角的十年」；小限=流年岁数提示，用来和流年对照。
  • 命主 / 身主：两颗「总代表」星名，不是真人；身宫标在你后天更在意的主题格。一眼看懂：命主身主是盘面的「总标题星」，不等于你本人；身宫格子常读作后天重心。
  • 十二宫题名速览：命宫=自己；兄弟=手足与同辈；夫妻=配偶感情；子女=子女与创作延伸；财帛=收入与花钱观；疾厄=体质与健康象（非看病）；迁移=外出、异地、外界眼光；仆役=朋友、同事、下属人缘；官禄=事业、学业、社会身份；田宅=房产与家庭根基；福德=心态、嗜好、精神享受；父母=长辈与上级缘分。一眼看懂：先记住「命宫财帛官禄夫妻」四个最常用的，其余当扩展包。
时辰怎么映射？（0–23 时 → iztro 0–12）

23点视为晚子时（序号 12），0 点视为早子时（序号 0）；其余每2 小时一时辰，与多数排盘一致。若您学的是「早晚子换日柱」另一套，请自行在纸上校正。

依赖与环境说明 · 若一点查询就闪退

需安装 py-iztro（依赖 pythonmonkey）。若安装失败请看 py-iztro。

重要：不要用 flask run 默认双进程热重载跑本服务；也不要在代码里开启 use_reloader=True。重载子进程常与内嵌 JS 引擎冲突，表现为一点「排盘」进程直接消失。本仓库 python app.py 已关闭 reloader；若你自行启动，请加 --no-reload 或等价选项。

宫位
这一行讨论的人生主题。标「命」=命宫；「身」= 身宫落此；「来因」= iztro 标记的来因宫（流派概念，可理解为因果/来处主题）。

宫干支
这一宫的天干+地支，像门牌，用来定星、论关系。

一眼看懂：两个字连写；干在上论生克，支定宫位方位序列。

主星
主角星；[庙/旺/得地/平/陷…]是亮度；化禄/权/科/忌是四化加持或加压。

一眼看懂：这一格谁唱主角；亮度=舞台条件；四化=临时剧情buff。

辅星
辅助星曜，补人事、补吉凶象。

一眼看懂：主角旁边的配角台词，有时比主角还抢戏，仍要整格一起看。

杂曜
小星、杂星一堆简称，古法里当细节注脚。

一眼看懂：密密麻麻的小标签，初学可略读，以后看书再对名词。

长生12
十二长生神煞名之一，读作「状态标签」，别当成医学。

一眼看懂：又是「长生沐浴冠带…」那种刻度词，和八字地势类似，别当体检报告。

博士12 / 将前12 / 岁前12
三套神煞/神将名号，用来细分年运、将星类象，当作文化符号即可。

一眼看懂：三套不同「神煞菜单」，列在一起是为了和专业盘对齐。

大限
这一宫当「十年大运主角」时对应的干支与大约岁数区间。

一眼看懂：这格宫位「承包」哪段十岁人生；换大限像换电视剧一季。
