# 四柱八字 Web · 全量排盘（子平基础 + 大运流年）

## 原理与实现

使用 **lunar_python**（6tail）的 `EightChar`、`Yun`、`DaYun`、`LiuNian` 等模块：

- 四柱、纳音、旬空  
- **天干十神**、**地支藏干**、**支藏十神**、**十二长生地势**  
- 胎元、胎息、命宫与身宫干支（库内八字法字段）  
- **大运**（含童限与多步大运）  
- **流年**：首步「实大运」内10 年流年干支示例  

## 使用注意

1. **必须选择性别**（大运顺逆依赖男女生年阴阳）。  
2. 未实现：真太阳时、早晚子时切换、刑冲合害、神煞大全、流月流日等。  

## 与紫微项目的区别

八字论命以**干支生克与运程**为主线；紫微以**宫位星曜**为主线。
