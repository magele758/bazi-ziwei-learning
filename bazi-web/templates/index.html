四柱八字推算

输入公历与性别，输出四柱、十神、藏干、地势、胎元命宫身宫参考，以及大运、流年（首步大运示例）。历法内核：lunar_python。

先用白话理解：八字在干什么？

古人把出生那一刻的「时间」翻译成一套汉字密码：用天干地支把年、月、日、时各写成两个字，一共八个字，所以叫四柱八字。你可以把它想成一张「时间身份证」——不是科学预测，而是传统命理里用来讨论性格、运势起伏的一种符号系统。

  • 年柱：常对应祖辈、大环境、早年外界对你的影响（流派不同解读会有差别）。一眼看懂：好比「你出生在什么年代、什么家庭氛围里」——别人先看到你的外包装。
  • 月柱：常对应父母兄弟、工作环境、社会角色——命理里叫「提纲」，分量很重。一眼看懂：好比「你从小被期待长成什么样、在社会上拿哪张名片」——很多格局论法都先看月。
  • 日柱：天干叫日元 / 日主，常被当作「你自己」的符号；地支也常与婚姻宫相关。一眼看懂：日干=剧本里的主角名字；日支=离你最近的那一格，也常用来聊亲密关系。
  • 时柱：常对应子女、晚年、事业结果、下半生的走向。一眼看懂：好比「人生后半场、收尾方式、和下一代/成果的关系」。

本页只做排盘与名词解释，不对人生下结论。不同老师对同一八字的解读可能不同。

算法说明：本程序具体怎么算出四柱？
  1. 公历 → 农历与节气：用开源库 lunar_python（作者6tail，与许多万年历同源思路）把您输入的公历时间换成农历日期，并按历法规则处理节气（八字的「月」通常跟节气走，不是农历初一换月）。白话：您填的是手机日历上的日子；程序再算出「节气月」——八字里的「几月」常指从某个节气到下一个节气这一段。
  2. 年柱：按该库规则以立春等为年界（不是正月初一）定年干支。白话：过年属相广告常按春节；很多八字排盘按立春换年，本程序跟库一致。
  3. 月柱：按节气月定月支，再用「五虎遁」类规则由年干推出月干。白话：先定「月支」（节气决定），再按口诀从年干推出「月干」，凑成两个字。
  4. 日柱：按连续日期循环六十甲子，算出当日干支。白话：像日历每天印一个「甲子、乙丑…」编号，出生日落在哪一天就抄哪两个字。
  5. 时柱：把 24 小时划成十二时辰，定时支；再用「五鼠遁」类规则由日干推出时干。白话：先定你是「子时、丑时…」哪一段，再按口诀从日干推出时干。
  6. 纳音 / 旬空：由四柱干支查表得出，是传统命理里的辅助信息。白话：不改四柱，只是多查两张传统附表，给每柱贴标签。

未做真太阳时（出生地经度校正）、未做「早子时 / 晚子时」手工分流；若您学习某派特殊规则，请以书本为准在纸面上校正。

名词小词典：结果里每一列是什么？
干支
一个字是天干（甲乙丙丁…），一个字是地支（子丑寅卯…），两个字合在一起记一个时间单位。

一眼看懂：像「姓+名」两个符号钉在一个时间点上；八字就是四个时间点各钉一对符号。

纳音
把一对干支再映射成「海中金」「炉中火」这类五行意象，用于传统命理的辅助分析。

一眼看懂：在「甲子、丙寅」这类干支代号之外，再贴一个文艺一点的五行外号，古法里当辅助看，不是改你的日干。

旬空
六十甲子一轮里总有「轮空」的两个地支，叫旬空。命理里常用来描述「虚」「待填实」之类意象，初学者知道是查表附注即可。

一眼看懂：想象六十个座位轮着坐，每轮总有两个座位本轮不点名——命理借这个讲「空、待补」，别当成科学结论。

五行计数
把四柱八个字的五行粗略数一数，只看「多寡」，不作吉凶判断——是否平衡要配合全局分析，不能单看个数。

一眼看懂：只是数「金木水火土」出现几次；某行多≠一定好，某行少≠一定坏，要看搭配。

排盘结果

这一页从上到下是什么？（超白话）

四柱八字
把出生时刻写成「年、月、日、时」四组汉字，每组一个天干+一个地支，共八个字。好比用密码记下你落地那一刻的「时间坐标」。

对照结果：页面上「四柱」那一行从左到右念完，就是八个字。

十神
以「日干」当「我」，看别的干支跟我是同类、生我、我生、克我、我克哪一种关系，给这种关系起名字（比肩、正印、七杀……）。用来描述人事、角色、矛盾，不是算命断定吉凶。

对照结果：在细表里找「天干十神」列——它回答「这一柱的天干，相对你自己是什么关系角色」。

藏干 / 支藏十神
每个地支里还藏着1～3个天干。把藏干再换算成十神，就看这一柱背后还有哪几种力量在掺和。

对照结果：「地支藏干」是里子，「支藏十神」把那些里子也换成十神语言。

地势（十二长生）
看日干落在这一支上，处于「长生、沐浴、冠带…」哪一站，像给能量标了个刻度：旺一点还是弱一点，是符号语言。

对照结果：细表最右一列；可理解成「日主在这一支上舒服不舒服」的粗略刻度，不是医学。

胎元、胎息
传统推演里「受孕前后」的辅助柱，用来补充看根气、渊源，现代多作参考。

一眼看懂：多出来的两柱「附录」，不是四柱本体；不懂可先跳过。

命宫、身宫（本页为八字体系里的两柱）
与紫微的命宫不是同一套算法；这里是八字流派里附加的干支组合，可作辅助读本。

一眼看懂：名字和紫微都叫命宫，但算法不同；不要拿八字命宫去对紫微命宫硬套。

大运
人生按十年一大段换一组「运程干支」。童限是还没进入第一步大运前的小段童年运。

对照结果：大运表里每一行对应一个十年主题包，「童限」是正式进场前的缓冲段。

流年
每一「公历年」对应一个干支，像当年的天气标签，常拿来和大运、原局一起看。

对照结果：流年表「公历年→流年干支」；只看某一年的代号，像给那年贴个标签。

纳音
给每一柱干支再贴一个「海中金」「炉中火」之类的意象名，古法里当辅助五行看。

对照结果：纳音表里「纳音」列，四柱各一行。

旬空
这一轮六十甲子轮空的那两个地支，传统里说「空」象，多指虚、待补、计划赶不上变化之类，初学者当备注看即可。

对照结果：纳音表最右「旬空」列；先当脚注，不必背。

大运

大运怎么读

  • 童限：还没正式进入第一步大运之前，用的「童年段」标记。白话：大运正式「开机」前的片头曲，有的盘会单独标出来。
  • 顺行 / 逆行：大运干支往前数还是往后数，由出生年阴阳与性别共同决定。白话：同样八字，男顺女逆或相反，规则是死的；您只要选对性别，程序会替您算。
  • 起运时间：从出生后多久「算正式进入第一段大运」，用年月日表示，古法近似值。白话：「几岁起运」不是心理学年龄，是排盘术语：从哪天起把第一段十年大运算到你头上。
  • 表头怎么读：序=第几步（第几个十年包）；大运干支=这十年的主题代号；年份=这十年落在哪些公历年；岁数=虚岁区间，和老黄历计岁一致。白话：把每一行想成「第N个十年套餐 + 套餐名字 + 大概哪几年」。
